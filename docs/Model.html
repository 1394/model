# static setConfig(cfg)

Parameters:
Name Type Description
cfg Object | Array

can be Object or Array of Objects if used multi-db configurations.
in multi-db config you should mark one db as default or define database name in each Model instance
elsewhere will be used first config in array mysql2 connnection config, https://github.com/sidorares/node-mysql2/blob/master/lib/connection_config.js
config.default {Boolean} set database config as default in multi-db configuration

 {database: 'db1', user: 'user', password: 'pwd', host: '127.0.0.1'},
 {database: 'db2', user: 'user', password: 'pwd', host: '127.0.0.1', default: true}
])

# with(scope, arg, …arg)

set scope from any Model methods or apply scope methods (all scopes stored globally in Model class and all instances has access to scopes)

Parameters:
Name Type Attributes Description
scope string

scope name
arg function

if function then Model.with set scope with name and scope function
arg arguments <repeatable>

View Source model.js, line 840

Examples
with arrow function : Model.with('active', (model,state) => model.where('active = ?', state)) and for use Model.with('active', 1)
with usual function : Model.with('active', function(model,state) {return model.where('active = ?', state)}) and for use Model.with('active', 1)
